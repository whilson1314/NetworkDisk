# NetworkDisk
基于Qt的C++项目，实现了网盘的基础功能，包括注册登录、好友系统、私聊群聊、文件操作、分享文件等功能，实现了网络通信、多线程等。

## 项目技术栈
主要编程语言：C++

开发平台：Qt Windows 5.15.2

设计特性：单例设计模式、网络通信
